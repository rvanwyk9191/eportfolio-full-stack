Website Architecture

Linux:

Linux is the host machine used. I am using AWS Lightsail to host my website.

Java:

Running on Java 8. No deprecated language is used, so an upgrade to Java 11 will happen seamlessly.

Spring:

Using Spring 2.3.1.RELEASE to run the website. Currently, I am using spring-boot-starter-thymeleaf and spring-boot-starter-web is used to process the website. Plans are to move all the content to a database, and include a login feature to update properties.

Thymeleaf:

Since Thymeleaf works seamlessly with Spring, I decided to use Thymeleaf as a processor for all MVC requests.

Bootstrap:

Bootstrap allows me to use predefined styling. I am not a stylist, so this saves me a lot of time.

HTML/CSS/JavaScript/JQuery:

HTML and CSS are returned after Thymeleaf processes the view. JQuery is a JavaScript language and is used by bootstrap. There are plans to do more integration with JQuery.
